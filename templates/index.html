Abstract: The Speech to Text project aims to create a system that can convert spoken words into written text and to develop a reliable system for accurate transcription of spoken language into text, benefiting individuals with impairments and enhancing communication across industries.. Here we are using the Google Cloud Speech Application Programming Interface (API), this allows converting audio to text and we are integrating the application with notepad . We are using the pynput library to simulate keyboard inputs, the speech recognition library to convert voice input into text, and the pywinauto library to interact with the Notepad application where we can also save the recognized speech as a file in notepad. Several steps are performed internally by this API for converting speech to text such as taking audio input, preprocessing, feature extraction, acoustic modelling, language modelling.

Introduction: The proposed system implements a voice-controlled interface for interacting with the Notepad application. The goal of this project is to enable users to control various functions of the Notepad application using voice commands, thereby providing a hands-free and intuitive method of interaction. This technology allows for the conversion of spoken language into written text, providing a more accessible and efficient means of communication. The development of speech recognition systems has been driven by the increasing demand for hands-free and intuitive interfaces for devices and applications. These systems have a wide range of applications, including voice assistants, transcription services, call center automation, and language translation. In this project, we will explore the development of a speech-to-text system using Google Cloud API integrated with notepad. The goal of this project is to create a system that can accurately and reliably transcribe spoken language into text, providing a valuable tool for individuals, as well as improving communication and productivity in various industries.
